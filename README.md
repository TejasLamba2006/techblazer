# TechBlazer

Team showcase website for STMicroelectronics TechBlazer program. Built with Next.js 14, React 18, Tailwind CSS v4, and Three.js.

## Getting Started

```sh
npm install
npm run dev
```

## Build

```sh
npm run build
```

Static export output goes to `out/`.

## Stack

- Next.js 14 (App Router, static export)
- React 18 + TypeScript
- Tailwind CSS v4
- Three.js + React Three Fiber (WebGL hero & tag sphere)
- Framer Motion (scroll animations, counters, 3D tilt)
- Lucide React + React Icons (icons)
